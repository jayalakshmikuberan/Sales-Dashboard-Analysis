Sales Dashboard Analysis (Power BI)

 Project Overview

This project focuses on building an interactive sales dashboard using Power BI to analyze business performance across different regions, product categories, and time periods.

The dashboard provides clear insights into key performance indicators (KPIs) and helps in data-driven decision-making.

 Objectives

* Analyze overall sales performance
* Compare revenue across regions and categories
* Track sales trends over time
* Identify key business insights

  Dashboard Features

  Total Sales KPI
  Total Profit KPI
  Total Quantity Sold
  Profit Margin (%)
  Sales Trend (Year-wise Line Chart)
  Sales by Category (Bar Chart)
  Sales by Region (Donut Chart)
  Interactive Filters (Slicers)



Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Visualization



Key Insights

* Sales show a consistent upward trend over time
* Technology category generates the highest revenue
* Profit varies across different product categories
* Regional performance highlights key business areas



Files Included

* `Sales_Dashboard.pbix` – Power BI dashboard file
* `dataset.csv` – Dataset used for analysis
* `dashboard_screenshot.png` – Dashboard preview





 Conclusion

This project demonstrates how raw sales data can be transformed into meaningful insights using Power BI dashboards, enabling better business decisions.
